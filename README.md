# CMake-Cpp-Cuda

A project for practicing CMake, Cpp, Cuda.

# Build

To build the project, you need to specify the location of nvcc:

```
-DCMAKE_CUDA_COMPILER=/usr/local/cuda/bin/nvcc
```
